# Running

## Install

Install Rust 2018 edition (if you don't have it already)

```
curl --proto '=https' --tlsv1.2 -sSf https://sh.rustup.rs | sh
```

## Build

```
cargo build
```

## Run

Run solution for a particular day
```
cargo run --bin day1
```
